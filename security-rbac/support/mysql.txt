




这里记录下在CentOS 7 上，安装配置MySQL80的步骤；


1下载yum源的安装包
yum localinstall https://repo.mysql.com//mysql80-community-release-el7-1.noarch.rpm

2yum安装
yum install mysql-community-server

3启动服务
service mysqld start

4查看状态
service mysqld status

这里写图片描述

5查看初始密码
grep 'temporary password' /var/log/mysqld.log

这里写图片描述

6登录 用临时密码
mysql -uroot -p
密码使用上面查到的


7更改密码
ALTER USER 'root'@'localhost' IDENTIFIED BY '你的密码';
密码策略默认需要设置复杂一点的

此时服务器上的安装配置已经完成，然后远程连接时发现并不能连接成功，原因可能是因为服务器3306端口没有对外开放；数据库用户没有远程连接登录权限；

8配置MySQL允许外部访问

进入mysql数据库 use mysql，修改mysql库的user表，将host项，从localhost改为%。%这里表示的是允许任意host访问，如果只允许某一个ip访问，则可改为相应的ip
GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY '密码' WITH GRANT OPTION;

刷新应用最新的配置
flush privileges

再次远程连接测试，连接成功

9如果不能在外部连接请检查系统防火墙设置，
